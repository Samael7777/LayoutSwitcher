3. Установка и отладка метода ввода
Метод установки системы 64 -бит отличается от 32 -битной системы. 64 -битные системы совместимы с приложениями 32 -битных систем. Метод ввода не может работать. 64 -битные приложения должны иметь метод ввода 64 -бит, а 32 -бит -программа должна иметь метод ввода 32 -бит.
Поэтому мы должны установить метод ввода в системе 64 -бит. Прежде всего, мы должны составить наш метод ввода DLL в 64 -бит и 32 -бит.
